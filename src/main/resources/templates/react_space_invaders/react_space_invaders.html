Overview: A tutorial on how to create a simple clone of Space Invaders in React. Credit to: https://cheesyprogrammer.com/2017/09/25/game-programming-using-javascript-react-canvas2d-and-css-part-1/ for the tutorial.

Completing the tutorial will provide a basic framework of this popular game where the player controls a movable ship and must destroy oncoming enemies. This version incorporates a score counter and proper collision detection.
