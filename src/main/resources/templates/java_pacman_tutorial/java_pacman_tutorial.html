Overview: A tutorial on how to recreate the arcade hit Pacman in Java.

Completing the tutorial will leave you with an approximate clone of this famous video game hit. Pacman is controlled by the arrow keys, and the ghosts will end the game if they catch him.

With a few adjustments the game speed is able to be increased, with the overall 'theme' being customized towards alien abduction. The player now controls a flying saucer which emits smoke trails, scouring the board to collect spawned cows.
