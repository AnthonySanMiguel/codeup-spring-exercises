Overview: A tutorial on how to recreate the classic game Snake in Java. Credit to: http://zetcode.com/javagames/snake/ for the tutorial.

Completing this tutorial will provide you with a great starting template to dive into and tweak to your liking.

With a few adjustments the game speed is able to be increased, with the overall 'theme' being customized towards alien abduction. The player now controls a flying saucer which emits smoke trails, scouring the board to collect spawned cows.
