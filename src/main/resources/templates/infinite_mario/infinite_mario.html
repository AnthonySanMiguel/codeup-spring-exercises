Overview: This open source code, by the author's admission, is a clone of the original Java version of Infinite Mario ported by non other than Markus Persson (aka "Notch" of Minecraft fame). This version runs on JavaScript for web browsers and uses HTML5. Credit to: https://github.com/robertkleffner/mariohtml5 for the open source code.

This clone of a port still astoundingly captures the look, feel, and sound of a genuine Super Mario game.

Levels are dynamically generated, along with enemy placement, coins, breakable bricks, and obstacles.
