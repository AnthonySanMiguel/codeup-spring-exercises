Overview: A tutorial on how to create a memory game using React. Credit to: https://www.golangprograms.com/react-js-projects-for-beginners/memory-game-in-react-js.html for the tutorial.

The game presents the player with images of celebrities. The player must click each picture only once, avoiding repeat clicks. The images will shuffle each time. If the same image is clicked a second time, the score resets.

Of course, feel free to experiment once completed and change the colors, ghost pngs, and even design your own maze!
