Overview: A paired-programming project using an AJAX call to simulate pinging a server database, retrieving the data, and presenting it in a user-friendly format. Users may add, edit, or delete entries, which dynamically update. Credit to: https://github.com/GuillermoDeLeon and https://github.com/johnnymdelgado12 for their contributions.

Users are greeted with a tabular display of movies available in the database, to include the entry ID, title, genre, and rating for each.

At the bottom of the table, users have the option to Add new entries, or Edit existing ones.

As an example, if we add a movie to the list (The Terminator)...

After clicking the "Add a Movie" button, we see the table refresh and the new entry populate on the list.

By clicking the "Remove" button on the right of the entry (The Terminator), the user is greeted by a confirmation prompt.

Once confirmed, the table once again refreshes on click and the data is updated dynamically.
