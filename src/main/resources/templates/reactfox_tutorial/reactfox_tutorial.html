Overview: A tutorial on how to implement three.js models in React using 'react-three-fiber' to create a Star Fox inspired template.
Credit to: https://codeworkshop.dev/blog/2020-03-31-creating-a-3d-spacefox-scene-with-react-three-fiber/ for the lesson.

Upon completion of all three tutorials, you are now able to run a basic wireframe of a shooter-style game.

After a little bit more experimentation, you can add custom models, colors, and even textures! I attempted a more 'grounded' destroy-the-base game rather than blasting incoming asteroids in space.
